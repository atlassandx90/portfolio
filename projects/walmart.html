Executive Dashboard

Business Problem

Retail businesses generate massive sales data every day. Without proper analytics, identifying customer trends, regional performance and future demand becomes difficult. This project helps decision-makers monitor KPIs, forecast sales and improve profitability.

Dataset

  • Retail Transactions
  • Sales Records
  • Customer Data
  • Regional Sales
  • Product Categories
  • Historical Trends

Project Workflow

Data Collection

Imported retail sales dataset from Walmart transaction records and prepared it for analysis.

Data Cleaning

Removed duplicates, handled missing values, standardized columns and validated data quality using Python.

SQL Analysis

Wrote SQL queries to analyze sales, customer behaviour, product performance and regional trends.

Dashboard

Built interactive Power BI and Tableau dashboards for business stakeholders.
